Counting Occurrences

  • count the number of times a particular boolean expression is true

Counting Occurrences

count_o = 0
sentence = "To be or not to be?"
for letter in sentence:
    if letter == "o":
        count_o += 1
print("There were", count_o, "o's in the sentence")  # 4
  • count each occurrence of the letter "o"

Searching

  • know of a particular element exists in a set
  • keep a list of all matches

Searching

# find out if element exists in list
numbers = [3, 7, 4, 9, 2]
search_term = 4
found = False
for n in numbers:
    if n == search_term:
        found = True
        break
if found:
    print(search_term, "is in", numbers)
else:
    print(search_term, "is not in", numbers)
  • assume the search term isn't in the list
  • mark that the term is in the list
  • stop searching

Searching

Swapping

  • swap the value of one variable with another

Swapping

# initialize
x = 2
y = 3
print(x, "and", y)  # 2 and 3

# swap
temporary = x
x = y
y = temporary
print(x, "and", y)  # 3 and 2
  • most simple case

Swapping

numbers = [3, 7, 4, 9, 2]
print("Before swap:", numbers)  # [3, 7, 4, 9, 2]

# swap the first and second numbers
temporary = numbers[0]
numbers[0] = numbers[1]
numbers[1] = temporary
print("After swap:", numbers)  # [7, 3, 4, 9, 2]
  • swapping within a list

Minimum and maximum

  • find the highest and lowest value in a set

Minimum and maximum

# find minimum
numbers = [3, 7, 4, 9, 2]
lowest = 9999
for n in numbers:
    if n < lowest:
        lowest = n
print("The lowest number in", numbers, "is", lowest)
  • initialize the lowest number found as some super high number
  • this guarantees any number in the list will be lower than it
  • update running lowest when a smaller number is found

Minimum and maximum

# find longest
animals = ["ape", "bison", "crocodile", "dog", "elephant"]
longest_word = ""
for animal in animals:
    if len(animal) > len(longest_word):
        longest_word = animal
print("The lowest word in", animals, "is", longest_word)
  • initialize the lowest number found as some super high number
  • this guarantees any number in the list will be lower than it
  • update running lowest when a smaller number is found

Testing

  • make sure that functions work as expected
  • try to find inputs that break your function
  • alter function so it no longer breaks
  • rerun to make sure the changes didn't break anything else

Testing

def are_equal(name, first, second):
    """Tests if two inputs are equal. Requires test name, first value to compare,
    and second value to compare.
    e.g. are_equal('To integer from string', int('5'), 5)
    """
    if first == second:
        print("Pass:", name)
    else:
        print("Fail:", name, ":: Expected:", second, ":: Received:", first)
  • simple method to test if two inputs are equal
  • save as "test.py" for use as a module
